# ReplayPack v0.2.0

Initial public npm release.

## What Ships

- `replaypack capture`
- `replaypack verify`
- GitHub composite Action support
- account-access wrong/fixed demo
- ProofBench evidence
- private public-repo trial evidence
- security note for trusted capsules and CI isolation

## Evidence

- ProofBench: 10/10 benchmark cases passed
- Visible-green wrong fixes rejected: 10/10
- Correct fixes accepted: 10/10
- False positives: 0
- Private public-repo trials: 7/7 selected public repositories passed
- Hosted GitHub CI: passing

## Known Limits

- Capsules execute proof and invariant commands; ReplayPack is not a sandbox.
- Public demand is not proven yet.
- External developer comprehension proof remains the next market gate.
