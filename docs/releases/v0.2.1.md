# ReplayPack v0.2.1

Patch release for public beta proof collection.

## What Ships

- one-command external developer trial: `npm run trial:external`
- structured external proof recorder: `npm run record-external-proof`
- external trial GitHub issue form
- readiness receipt validation for trial receipts, transcripts, npm, CI, dogfood, and live-agent proof
- expanded Claude Code full-generation evidence: 6/6 ReplayPack treatments verified vs 5/6 controls correct

## Evidence

- ProofBench: 30/30 benchmark cases passed
- AgentBench deterministic replay: 30/30 visible-only false-done outcomes prevented and recovered
- Claude Code full-generation trial: ReplayPack treatments 6/6 verified, controls 5/6 correct with 1/6 false-done
- Hosted GitHub CI runs `npm test`, `npm run trial:external`, package creation, and local Action verification

## Known Limits

- External developer proof is still the next readiness gate.
- ReplayPack executes proof and invariant commands; it is not a sandbox.
